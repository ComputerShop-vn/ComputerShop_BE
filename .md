Impl feature Create Product in ProductController follow by 3-layer architechture.
DTO request include: 
    - categoriesID
    - brandId
    - name
    - description
    - price
    - stockQuantity/ list serialNumber (user have two option, manage by stockQuantity or serialNumber, if stockQuantity is null, then list serialNumber is not null, and vice versa)
    - attributes (list attributeId, attributeValue)
    - images (list image url)
DTO response include:
    - productId
    - categoriesID
    - brandId
    - name
    - description
    - price
    - stockQuantity/ list serialNumber (user have two option, manage by stockQuantity or serialNumber, if stockQuantity is null, then list serialNumber is not null, and vice versa)
    - attributes (list attributeId, attributeValue)
    - images (list image url)
    Note: warp in ApiResponse include, code, message, result.
ALL entity relatetive: 
    - Product
    - ProductItem
    - ProductAttribute
    - ProductImage
    - Brand
    - Categories
    - Attribute
    - CategoriesAttributes
